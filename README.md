## head
#### 웹 페이지의 정보, 문서에서 사용할 외부 파일들을 링크할 떄 사용함.

## body
#### 웹 브라우저에 실제 표시되는 내용

## body에서 사용하는 기본적인 태그들
### - p
#### 하나의 문단을 만들 떄 사용
### - br
#### 출력할 내용을 입력 후 줄바꿈을 실행할 떄 사용
### - a
#### 다른 웹페이지나 외부 사이트 연결할 떄 사용
### - img 
#### 이미지 삽입할 때 사용
### - li
#### ul과 ol안에서 각 항목을 나열할 떄 사용

### - ul 
#### 순서가 없는 리스트를 만들 때 사용
